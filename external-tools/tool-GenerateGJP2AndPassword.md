This tool can generate the **GJP2** *And* **Password** fields, so you can change passwords in [PHPMyAdmin](https://www.phpmyadmin.net/) easily.
![image loading..](https://file.garden/ZtBrW_PK_QLwMwt1/dese%20ones.png)
Python download: https://file.garden/ZtBrW_PK_QLwMwt1/Create%20GJP2.py
How to use:
* Open command prompt in the same directory as the .py and type the command: pip install bcrypt.
* After it installs, type the command: Python "Create GJP2.py" (replace Create GJP2.py if you changed the name of the python).
* Enter your password. `(Everything is done locally on your device, so don't worry about it being hacked)`
* Edit the **password** and **GJP2** fields in phpmyadmin to the ones givin, (it's in the "accounts" field. ![image loading..](https://github.com/user-attachments/assets/96bf94aa-677d-4727-8268-0aa27040bac7)
* Login/Refresh login with the password you entered.
* This can be used to make passwords impossibly long, requiring character and length limit hacks nessecary to logging in (aka more security).  
`(Note: If pasting your impossibly long password didn't work with hacks, just paste it again as ctrl V replaces the text input normally)`
